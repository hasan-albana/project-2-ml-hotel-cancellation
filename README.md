# project-2-ml-hotel-cancellation

Proyek ini bertujuan untuk membangun model machine learning yang dapat memprediksi kemungkinan pembatalan pemesanan hotel.
Model dikembangkan sebagai alat bantu analisis untuk mendukung segmentasi risiko pembatalan dan pengambilan keputusan berbasis data.
